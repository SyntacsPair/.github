# 🧑‍🤝‍🧑 About Us

**SyntacsPair**는 프론트엔드와 백엔드 개발자 2명이 모여 결성한 개발 팀입니다.

> *"Syntax(구문) + Pair(짝)"*  
> 서로 다른 영역의 코드가 만나 하나의 완성된 서비스를 만들어갑니다.

- 🎯 기획 단계부터 함께 논의하며 방향을 설정합니다
- 🛠️ 각자의 전문 영역에서 최선의 결과물을 만듭니다
- 🤝 코드 리뷰와 피드백을 통해 함께 성장합니다
